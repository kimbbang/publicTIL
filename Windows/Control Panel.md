# Windows

## Settings, the Control Panel, Task Manager

### Settings 
: 설정(윈두우버튼 > 톱니바퀴 모양)   

### Control Panel 
: 제어판(검색 > 파란색 네모난 계기판 모양)   

### Task Manager   

* right-clicking the taskbar > click Task Manager(작업관리자) 또는   
* Ctrl + Shift + Esc
