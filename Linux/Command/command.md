









## whoami	
Find out what user we're currently logged in as!
```
tryhackme@linux1:~$ whoami
tryhackme
```



## echo	
Output any text that we provide
```
tryhackme@linux1:~$ echo "Hello Friend!"
Hello Friend!

tryhackme@linux1:~$ echo Hello
Hello

$ echo "'Hello world'"
'Hello world'

$ echo '"Hello world"'
"Hello world"
```

## ls

root@ip-10-10-250-80:~# ls
