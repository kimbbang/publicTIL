# FDDI
Fiber Distributed Data Interface

파이버 분산형 데이터 인터페이스(Fiber Distributed Data Interface, FDDI)는 근거리 통신망의 데이터 전송 표준이다. 광섬유 분산 데이터 인터페이스, 광섬유 분산 데이터 접속 방식이라고도 한다.

현재는 사용 안 함
